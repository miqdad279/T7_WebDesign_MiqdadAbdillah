~ HTML 5 ~

Introduction

HTML5 adalah sebuah bahasa untuk menstrukturkan dan menampilkan isi dari World Wide Web, sebuah teknologi inti dari Internet. HTML5 adalah revisi kelima dari HTML dan hingga bulan Juni 2011 masih dalam pengembangan.

Tujuan dibuatnya HTML5

  • Mengurangi kebutuhan untuk plugin eksternal (seperti Flash).
  • Penanganan kesalahan yang lebih baik.
  • Lebih banyak markup untuk menggantikan scripting.
  • HTML5 merupakan perangkat mandiri.

Dukungan browser yang support HTML5

  • Chrome
  • Safari Apple v.4+
  • Internet Explorer 9
  • Opera
  • Maxthon
  • Firefox 4, 5, 6 dan seterusnya

Fitur baru dalam HTML5

  • Unsur kanvas untuk menggambar.
  • Video dan elemen audio untuk media pemutaran.
  • Dukungan yang lebih baik untuk penyimpanan secara offline.
  • Elemen konten yang lebih spesifik, seperti artikel, footer, header, navigation, section.
  • Bentuk kontrol form seperti kalender, tanggal, waktu, e-mail, URL, search.

Deklarasi pada HTML5

  • Menggunakan deklarasi DOCTYPE
    <!DOCTYPE html>
  • Menggunakan deklarasi character encoding (charset)
    <meta charset="UTF-8">

HTML5 Example


<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Title of document</title>
<head>

<body>
content of the document
</body>

</html>

New HTML5 Element

  • semantic elements : <header>, <footer>, <article>, and <section>.
  • attributes dari form elements : number, date, time, calendar, and range.
  • graphic elements : <svg> and <canvas>.
  • multimedia elements : <audio> and <video>.

New Semantic Elements in HTML5

  • Banyak website yang menggunakan html code seperti : <div id = “nav”> <div class=“header”> untuk mengindikasikan mana navigation dan header.
  • HTML5 memiliki struktur yang telah di definisikan.
  • <article> : mendefinisikan artikel
  • <aside> : mendefinisikan konten di sisi kiri/kanan
  • <details> : mendefinisikan detail
  • <figcaption> : mendefinisikan caption pada gambar
  • <figure> : menentukan konten
  • <footer> : mendefinisikan footer
  • <header> : mendefinisikan header
  • <main> : mendefinisikan konten utama
  • <mark> : mendefinisikan marked text
  • <nav> : mendefinisikan navigasi link
  • <section> : mendefinisian sesi dari konten
  • <summary> : mendefinisikan judul untuk detail
  • <time> : mendefinisikan tanggal/waktu
